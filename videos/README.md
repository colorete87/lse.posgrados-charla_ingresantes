# YouTube Video Downloader

This folder contains a Python script to download YouTube videos in MP4 format.

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the download script:

```bash
python download_videos.py
```

The videos will be downloaded to the `videos` folder with their original titles as filenames.

## Requirements

- Python 3.6+
- yt-dlp library
